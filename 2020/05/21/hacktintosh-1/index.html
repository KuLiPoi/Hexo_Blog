卡特琳娜的调教日记 0x1
发表于 2020-05-21| 更新于 2020-05-21|Hackintosh
阅读量:

这什么鬼名字啊
卡特琳娜是哪位啊
你为什么要调教她啊
你是不是有怪癖啊

你错了 Catalina是美国西海岸靠近洛杉矶22英里的一个小岛，当然也可以是一个人名。
他也可以是 苹果公司的一个新系统代号为 Catalina

其实黑苹果早就装好了，最近又帮我朋友装了一台。
授人以鱼不如授人以渔，就趁机写个教程吧。
「本文均是自己装机的个人理解，如果那里说的不对，欢迎指正」H

准备

  • 电脑硬件查看

    • 显卡
      如果你的显卡是N卡，那么你装黑苹果就只能止步于 High Sierra，因为老黄已经不再为10.13之后的系统更新驱动了，具体为什么，你可以参阅这篇文章，如果你的显卡是A卡，则可以参阅黑果小兵大神的A卡免驱列表。我们常见的 5799xt / vega64 / vega56
      / rx 580 等均可免驱。

    • 网卡
      一般台式鸡的板载网卡都是免驱的，如果你是笔记本，则可以参阅这篇文章黑苹果机型维护清单。如果你是外置网卡，则需要看官方有没有对应的Macos驱动或者可以Google Wireless USB Adapter Clover

    • 硬盘
      我这里推荐是单独购入一块120g的固态硬盘，淘宝某些杂牌也就不到100块，你装起来开心又省力，何乐而不为。这里不再赘述如何将Mac和Windows装入同一个硬盘，可能后面会另外开文补充喔。

    • 其他

      1. 准备一个8G以上的U盘「我的意思是16G」
      2. 从网上下载Etcher
        Tips：经过我的测试发现，目前最新的及1.12以上的版本均不能为Catalina制作安装镜像，会出现Checksum does not mtch for range 的错误，请使用1.12远古版本，我目前装了3台机子，都使用的远古版本的1.12都没有问题，如果你担心会出问题，那么可以在Linux环境下使用Etcher最新版，测试不会报错。
      3. 系统镜像 截止至 2020年5月21日，我能找到的最新的镜像
        Catalina [迅雷离线下载] [HTTP下载] [奶牛快传]
        HighSierra [迅雷离线下载] [HTTP下载] [百度云]
      4. 在Windows下准备好 Diskgenius
      5. 你的 🧠

制作安装镜像

好像没啥好说的，打开Etcher，选择U盘，点Flash即可。

Bios设置

不同主板的设置大同小异，寻找对应的关键字即可

  • 检查 Secure Boot => Disable
  • 检查 Fast Boot => Disable
  • 检查 CSM(Compatibility Support Module) => Enable
  • 检查 OS Type => Other OS
  • 检查 VT-d => Disable
  • 检查 VT-x => Disable
  • 检查 Sata Mode Selection => AHCI
  • 检查 XHCI Hand-off => Enable

安装系统

  • 在Clover主界面选择 Boot macOS Install from Install macOS Catalina,回车即可
    「如果你安装的是 Catalina，请在安装过程的每次开机的时候移动到Option [小齿轮] => Boot Args 移动到最后添加 agdpmod=pikera
  • 根据你电脑配置的不同，可能会等待2-5分钟不等，然后会进入安装界面，然后看到 MacOS实用工具
    选择 磁盘工具 => 选中你要安装黑苹果的固态 => 抹掉 => 选择 格式: APFS 方案: GUID 分区图 => 抹掉
  • 点击完成之后离开磁盘工具 选择安装MacOS ，然后一路继续即可，期间可能会重启两次到三次

即将完成

一路同意加继续即可 AppleID 不登录 直至进入系统。至此，你的黑苹果应该已经可以开始使用了，目前黑苹果算是仅仅可用的状态，如果你想得到和白果一样的体验。有句话怎么说的来着「路漫漫其修远兮 吾将上下而求索」

总结

装逼结束，我觉得说的还算轻巧，安装过程不会超过20分钟，黑苹果难在后期的优化，如果将黑果做到和白果一样的体验。我后面会再出一些教程，来补充说明后面的优化和驱动
